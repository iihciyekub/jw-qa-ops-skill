## 结构化知识摘要（由 PDF 精确提取）

### 1. 流程目标

- 对半成品在生产中关键节点进行查货，确认款式、尺寸、车工及其他规格符合要求。

### 2. 前置资料（查货资料）

- 款式资料
- 色咭
- 物料咭
- 批办
- 合同
- 有关更改资料
- Q.C File / 系统上传文件

### 3. 流程活动与规则

- `3.2.1 收集查货信息`
- 品质保证部定期收集采购部/供应商发出的半成品查货申请，安排 QA 行程并通知工厂落实安排。
- 于中期查货前，QA 需核对查货资料是否齐备，不齐备时须与采购部相关组别跟进。

- `3.2.2 执行查货工作`
- 采购部澳洲组、外客组订单：执行半成品查货。
- 采购部中国组订单：数量大于 700 件时执行半成品查货；700 件（含）以下按工厂类别规则执行半成品或成品查货。
- 对非免检工厂、小型工厂、现货/库存工厂，若连续 3 张单一次查货合格率未达 100%，进入不多于 3 个月整改期；整改期内执行半成品及成品查货，整改后按规则恢复。

- `3.2.3 跟进改善事项`
- 可改善问题：下一次查货时跟进并记录改善结果。
- 不可改善问题：在报告中注明问题占比，联络采购部商讨解决方案，必要时提交样办覆核并通知工厂处理结果。

### 4. 时点与范围

- 一般在开裁后、车缝中或包装前执行半成品查货。
- 如时间不足，最低限度需在包装前完成一次查货。

### 5. 抽样与检查规则

- 不同颜色中，每色抽取 3 至 4 件，每次不少于 20 件。
- 配合客户需求抽办度尺码，每码全度不少于 3 件，且需齐色。
- 每次查货至少抽 1 件并填写 `呎碼表：F-PACIQADT-013`。
- 检查项包含量尺、对款及其他规格（如嘜头位置等）。
- 各项瑕疵分类及接受准则参考《品质标准》。
- 半成品查货原则上不判定合格与否，重点记录问题及改善要求，并在 eTDS 查货系统完成报告。

### 6. 流程管理指标（KPI）

- 准时完成率
- 准时完成查货单数
- 计算公式：`准时完成率 = 准时完成查货单数 / 查货总单数 x 100%`
