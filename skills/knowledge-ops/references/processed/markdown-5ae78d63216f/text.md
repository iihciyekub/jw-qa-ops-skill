## 结构化知识摘要（由 PDF 精确提取）

### 1. 流程范围

- `3.3a` 成品查货（正常订单，除联营业务/联营款订单）
- `3.3b` 成品查货（联营业务/联营款订单）

### 2. 前置资料

- 正常订单（3.3.1a）
- 款式资料（中/英文制单或客人办单）
- 色咭
- 物料咭
- 批办
- 合同
- 有关更改资料
- 查货资料（Q.C File/系统上传文件）

- 联营业务/联营款（3.3.1b）
- 查货通知
- 查货资料（货号、数量、审款细节图、码数和颜色）
- 采购部每月抽取联营业务不少于 20% 上新款数、联营款 100% 款数作为查验订单

### 3. 流程活动与规则

- `3.3.1a 收集查货信息`：质量管理部门收集查货申请、安排 QA 行程并落实工厂查货安排；资料不齐备时须跟进采购部。
- `3.3.1b 收集查货信息`：按月接收联营查货申请，到指定仓库或地点执行抽查。

- `3.3.2a 执行查货工作（正常订单）`
- 查货时段：每单货品在工厂最少完成 80% 或 100% 装箱，或按客人要求后进行抽箱查验。
- 抽样：按客人要求；若无特别要求，按公司《品质标准》抽样，并按颜色数量比例分配抽查数。
- 量尺：按需求抽办度尺码，每码全度不少于 3 件且齐色不少于 1 件，并填写 `F-PACIQADT-013`。
- 对款及规格：布料、布色、款式、车工、洗水效果/手感、整洁度、熨工、尺寸、嘜头、撞钉、钮、吊牌、包装、箱嘜等。
- 合格判定：按抽检接受/不接受水平及《品质标准》执行；免检工厂由工厂按本组织标准自检并判定。
- 权责：仅限公司每月批签“认可信责权限名单”人员签署并决定是否合格去货。
- 报告：在 eTDS 查货系统（B/S）编制和提交成品查货报告；免检工厂合格报告由工厂录入并由口岸主管批签。
- 其他：每张订单由 QA 抽取船头办交工厂寄采购部。

- `3.3.2b 执行查货工作（联营业务/联营款）`
- 抽样标准：按 `QS-PACIQADT-002` 中 `AQL 2.5 II（正常）` 抽查表。
- 查货地点：联营仓库或指定地点。
- 查验范围：仅执行成品外观品质查验（布料、车缝、洗水、包装、染织缝、印花/印嘜/烫花/绣花疵点）。
- 联营业务/联营款只执行成品查验，不执行半成品查验；查验结束无需抽取船头办。
- 报告与归档：QA 在 eTDS 录入查货报告与疵点图片；公司各部门可查阅。
- 口岸抽查：可按采购申请或不定期随机抽查出货口货品并开具报告。
- 备货点数/复核：联营上架货品需 100% 复核（到仓抽查+样办复核），抽查比例不少于 20%。
- 权责与考核：仅授权名单人员可签署；每月对联营品控/自查员进行考核，并开展质量会议与整改建议。

- `3.3.3a 跟进改善事项（正常订单）`
- 可改善问题：可翻修且不延期时，要求翻修并复检。
- 不可改善问题：记录不接受事项及占比，联络采购部商讨解决方案，必要时提交样办覆核并跟进结果。

- `3.3.3b 跟进改善事项（联营业务/联营款）`
- 可改善问题：不影响上架销售时，要求翻修并复检。
- 不可改善问题：QA 及时联络采购部协商处理，必要时提交问题样板或图片并跟进结果。

### 4. KPI 指标

- 准时完成率
- 一次查货合格率（适用非免检工厂）
- 公式 1：`准时完成率 = 准时完成查货单数 / 查货总单数 x 100%`
- 公式 2：`一次查货合格率 = 一次查货合格单数 / 查货单数 x 100%`
